Raciocínio Algorítmíco

A matéria de Raciocínio Algortimíco é ministrada pelo professor Emerson Paraíso, nessa matéria aprendemos sobre logica de programação em portugol que futuramente foi substituída pelo Python 3. A disciplina é acompanhada de 2 projetos que foram produzidos em duplas, o primeiro a ser feito foi o pyEstar, a idéia do projeto era "reproduizir" o aplicativo Estar que tem como objetivo gastar créditos para ocupar uma vaga que é destinada ao Estar que foi lançado recentemente. O segundo projeto a ser feito foi o pyVacina, e nele devíamos implementar um programa para controlar, agendar e verificar as vacinas tomadas pelo usuário.

    Modelagem de Fenômenos Físicos

    A matéria de Modelagem de Fenômenos Físicos é ministrada pelo professor Orlando, nessa matéria aprendemos sobre logica de programação em Python 3 no Anaconda 3. Inicialmente começamos pelo estudo de comandos e funções básicas no JupyterLab, dentro das 27 aulas do semestre fomos evoluindo até chegar ao nível da disciplina de Cálculo I (Derivadas e Integrais). Ao longo do semestre fizemos duas provas, uma sobre Limites e outra sobre Derivadas e Integrais. Todos os exercícios foram individuais.

    Experiência Criativa

    A matéria de Experiência Criativa é ministrada pelo professor Luiz Pavão e pelo professor Eduardo Lino, nessa disciplina tivemos 3 projetos, o primeiro projeto foi construir um jogo na engine Construct 2/3, esse projeto foi realizado em grupos de até 4 pessoas. Meu grupo construiu um jogo no qual o objetivo era avançar nas fases(Exemplo: Mario) para salvar o Pavão Milenar. No segundo projeto tivemos que implementar um programa no Processing 3 em Java, esse projeto era individual e eu fiz um programa educacional sobre a flauta doce, o projeto ensinava a tocar duas músicas, disponibilizada um teclado virtual com a flauta e algumas imagens. O projeto 3 era o desenvolvimento de um aplicativo com 10 ferramentas do Visual Studio utilizando o Windows Forms
